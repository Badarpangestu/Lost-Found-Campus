# Lost & Found Campus

## Deskripsi
Lost & Found Campus adalah aplikasi web untuk membantu mahasiswa melaporkan barang hilang dan barang ditemukan di lingkungan kampus. Aplikasi ini menggunakan Google Sheets sebagai database dan Google Apps Script sebagai API.

## Fitur
- Tambah laporan barang hilang/temuan
- Menampilkan daftar barang
- Edit data
- Hapus data
- Pencarian berdasarkan nama barang

## Teknologi
- HTML
- CSS
- JavaScript
- Google Apps Script
- Google Sheets
- GitHub Pages

## Cara Menggunakan
1. Isi formulir laporan barang.
2. Klik **Simpan Data**.
3. Data akan tersimpan ke Google Sheets.
4. Gunakan tombol **Edit** untuk mengubah data.
5. Gunakan tombol **Hapus** untuk menghapus data.
6. Gunakan kolom pencarian untuk mencari barang berdasarkan nama.
